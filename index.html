fhost¶

fhost is a Python module that helps you to manage your /etc/hostnames easily.

Provides an API based on a CLI interface to add, remove and list maped local hostnames with IP’s that follows the standards of Hosts file.

At this moment it only runs in Linux, but we plan to write a port for OSX and Microsoft Windows.

It is licensed under the BSD three-clause license.
